HTTP (HuperTextTransferProtocol). Клиент-Серверное взаимодействие.
Коды состояние ответа:
1.. Информационные ответы.
2.. Ответ о успешно выполнение кода.
3.. Ответ о перенаправление.
4.. Ошибка со стороны сервера.

Запуск Django 5.1.1
1. Создаём папка в VSCode.
2. Создаём виртуальное окружение. python -m venv venv
3. Активируем виртуальное окружение. ./venv/scripts/activate
4. Скачиваем Django.    pip install Django
5. Создаём проект с именом core.    django-admin startproject core . (пробел и точка, чтобы не дублировать папку)
6. Миграция. python manage.py migrate.
7. Запуск.      python manage.py runserver
8. И запускаем http://127.0.0.1:8000/
9. Далее создаем суперюзер и тд.
